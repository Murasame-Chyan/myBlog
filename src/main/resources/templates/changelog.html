06-29

UI 增强合集

新增分享卡片生成、全息徽章展示组件、通用确认对话框；实现 JWT 静默续期机制；安全图片代理防止 SSRF；全局 CORS 跨域支持；升级 Bootstrap 至 5.3.3 并引入 spring-boot-devtools 热更新。

为创作者提供数据洞察面板：GitHub 风格 SVG 发文热力图、标签使用统计饼图、阅读/点赞/评论趋势图、最受欢迎文章排行；含骨架屏加载态。

06-02

签到升级系统

每日签到获取经验值与连续签到奖励；等级系统含进度条与等级徽章（绿→蓝→黄→橙→红→粉→紫渐变色）；成就徽章解锁与展示墙；博客封面图上传功能。

新功能
06-01

全息徽章卡片

为成就徽章实现 3D 全息卡片效果：镭射箔纹理、彩虹光泽、指针驱动倾斜变换（弹簧物理缓动）、缩略图与模态框交互。

新功能
05-27

JWT 无状态认证体系

全面迁移至 JWT 无状态认证：access token + refresh token 双令牌机制，refresh token 通过 HttpOnly Cookie 传输防止 XSS 窃取；authFetch 前端封装自动 401 续期；彻底禁用 JSESSIONID 与 Servlet 会话追踪，实现 STATELESS 安全策略。

新功能
05-27

小游戏页面 + GitHub 并行获取

新增休闲小游戏页面（easysurvivor）；GitHub 个人信息面板实现并行数据获取（热力图、成就、仓库列表），优化加载速度。

新功能

Redis 点赞系统
